codebuild-ios-mcp — AWS Architecture

Agent-driven iOS build & test on AWS CodeBuild macOS, exposed as four MCP tools through a Bedrock AgentCore Gateway. Optional VPC mode reaches internal Nexus & validation services.

1Agent → Gateway — SigV4-signed MCP over HTTPS (AWS_IAM) 2Gateway → Lambda — assumes GatewayInvokeRole, invokes the tool handler 3Lambda → CodeBuild — StartBuild; reads summary.json + presigned URLs from S3 4Build → internal — fleet reaches Nexus & validation services over the VPC

⚠ Cost: the MAC_ARM reserved fleet bills continuously (~$25–30/day/instance, ~24 h min lease).  •  The VPC, its endpoints, and the corporate-network tier are the optional mode (omit for public-egress builds).  •  The Gateway is created by register-gateway.sh (no IaC resource yet).
