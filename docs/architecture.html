Data flow

①Agent → Gateway over SigV4-signed MCP; gateway assumes its role and invokes the Lambda.
②Lambda starts the build, then reads summary.json + presigned artifact URLs back from S3.
③Build runs on the macOS fleet, writes results to S3; agent polls until done, drills into failures.
VPCOptional: fleet joins your VPC to reach Nexus + internal services; endpoints keep AWS reachable.
Cost: reserved macOS fleet bills continuously (~$25–30/day/instance, ~24 h min lease). Tear down when idle. Regions: us-east-1/2, us-west-2, eu-central-1, ap-southeast-2. Deploy: cdk deploy → register-gateway.sh.
